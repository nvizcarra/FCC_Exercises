#02 Create a Complex JSX Element

## IMPORTANT

El JSX anidado tiene que retornar un solo elemento.
Luego éste elemento puede contener todos los demás elmentos anidados.

> const JSX = <div>
>                <h1>Hola</h1>
>                <p>Saludos</p>
>                <ul>
>                    <li>Item 1</li>
>                    <li>Item 2</li>
>                    <li>Item 3</li>
>                </ul>
>            </div>
