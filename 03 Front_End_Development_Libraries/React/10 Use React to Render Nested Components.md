# Use React to Render Nested Components

Hay otras formas de unir componentes con React.

La combinación de componentes es la característica más poderosa de React.

Pensar en la UI en términos de componentes ayuda a separar el código de la UI del de la lógica. Ésto simplifica el desarrollo y mantenimiento de proyectos complejos.

## Ejercicio 
There are two functional components defined in the code editor, called TypesOfFruit and Fruits. Take the TypesOfFruit component and compose it, or nest it, within the Fruits component. Then take the Fruits component and nest it within the TypesOfFood component. The result should be a child component, nested within a parent component, which is nested within a parent component of its own!

        <li>Apples</li>
        <li>Blueberries</li>
        <li>Strawberries</li>
        <li>Bananas</li>
      </ul>
    </div>
  );
};

  return (
